## 📖 概要

Sequoia Capitalのパートナー、Julien Bekは言った。**「次の1兆ドル企業は、サービス企業の"顔"をしたソフトウェア企業になる」** と。

SaaSは死んでいない。分裂している。

シートベースの課金モデルは崩壊を始めた。Klarna は700人の顧客対応を25人分のAIに置き換え、Salesforceは「Agentforce」で自社SaaSの存在意義を自ら問い直している。$2兆の時価総額がSaaS市場から消失し、500席のライセンスが50席に圧縮される世界が、もう始まっている。

だが、本当の構造変化はその先にある。

ソフトウェアの市場規模は$1兆。サービスの市場規模は$6兆。AIが「ツール」を売る時代から「仕事の完了」を売る時代に移行したとき、TAMは6倍に拡大する。Sierra（$10B評価）、Harvey（$11B評価）、Mercor——すでにこの構造転換を体現するパイオニアたちが出現している。

**本書は、SaaSの死亡宣告ではない。AIが産業構造をどう書き換えるのか、その設計図だ。**

Sequoia Capital、BCG、Gartner、McKinseyの一次情報と、世界のパイオニア企業の実装事例を横断分析しながら、「次の1兆ドル企業」が満たすべき条件を構造化する。
